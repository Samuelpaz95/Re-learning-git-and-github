## Git configuration
- View git config
```bash
git config --list
```
- View config origin
```bash
git config --list --show-origin
```
- Config user
```bash
# user fullname
git config --global user.name "Willy Samuel Paz Colque"
# user Email
git config --global user.email "samuelpaz243@gmail.com"
```
